Prompt inicial
Crie um MVP simples de uma API REST com foco em sustentabilidade.

Contexto:
Este projeto é um exemplo educacional para o Bootcamp GitHub Copilot – Código na Prática, uma iniciativa da DIO em parceria com o GitHub.

Requisitos técnicos:
- Usar Node.js com Express
- Criar uma API REST funcional
- Não utilizar banco de dados, apenas dados em memória
- Implementar operações básicas de CRUD
- Manter o código simples e fácil de entender
- NÃO adicionar documentação, comentários ou README
- NÃO criar testes automatizados
- O objetivo inicial é apenas ter a API funcionando

Domínio do projeto:
A API deve gerenciar iniciativas de sustentabilidade, como:
- ações ecológicas
- metas de redução de carbono
- programas de reciclagem
- projetos de impacto ambiental

Estrutura esperada do projeto:
- src/app.js
- src/routes/
- src/services/

Saída esperada:
Forneça os arquivos de código necessários para que o projeto funcione localmente.
